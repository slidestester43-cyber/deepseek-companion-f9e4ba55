# 54 Global Afrikan — Netlify Contact Form

A standalone, brand-matched contact form that submits to **Netlify Forms** (no backend required).

## Deploy in 30 seconds

### Option A — Drag & drop
1. Go to https://app.netlify.com/drop
2. Drag the entire `netlify-form/` folder onto the page
3. Done — your form is live

### Option B — From Git
1. Push this `netlify-form/` folder as the root of a repo (or set "Base directory" to `netlify-form`)
2. Connect the repo in Netlify
3. Build command: *(leave empty)* · Publish directory: `.`

## Where do submissions go?
Netlify dashboard → **Forms → contact**. Enable email notifications under Form settings.

## Files
- `index.html` — the styled form
- `thank-you.html` — success page (form redirects here on submit)
- `netlify.toml` — minimal config
